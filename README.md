# Machine Learning Repository

This repository contains implementations, experiments, and learning materials for **Machine Learning**, developed as part of academic study and practical exploration. It combines theoretical understanding with hands-on coding using Python.

---

## Objective

To build a strong foundation in machine learning by implementing algorithms, analyzing data, and understanding model behavior through experimentation.

---

##  Repository Structure (Branches)

- **`main`** → Portfolio-ready projects  
  - Clean implementations  
  - Structured code  
  - Reproducible results  
  - Focus on real-world applications  

- **`notebooks`** → Learning and experimentation  
  - Jupyter notebooks  
  - Step-by-step explanations  
  - Data exploration (EDA)  
  - Model prototyping  

---

##  Topics Covered

- Supervised Learning  
  - Linear Regression  
  - Logistic Regression  
  - k-Nearest Neighbors (k-NN)  
  - Support Vector Machines (SVM)  
  - Decision Trees and Random Forests  

- Unsupervised Learning  
  - K-Means Clustering  
  - Dimensionality Reduction (PCA)  

- Model Evaluation  
  - Train/Test split  
  - Cross-validation  
  - Metrics (Accuracy, Precision, Recall, F1-score)  

- Data Preprocessing  
  - Normalization and scaling  
  - Handling missing data  
  - Feature selection
