# Machine Learning - Notebooks

This branch contains **Jupyter notebooks** focused on learning, experimentation, and step-by-step implementation of machine learning concepts.

---

## Purpose

The `notebooks` branch is dedicated to:

- Understanding core machine learning concepts  
- Exploring datasets (EDA)  
- Prototyping models  
- Visualizing results  
- Testing ideas before moving them to production-ready projects  

---

## Contents

The notebooks cover topics such as:

- **Supervised Learning**
  - Linear Regression  
  - Logistic Regression  
  - k-Nearest Neighbors (k-NN)  
  - Support Vector Machines (SVM)  
  - Decision Trees  

- **Unsupervised Learning**
  - K-Means Clustering  
  - Principal Component Analysis (PCA)  

- **Model Evaluation**
  - Accuracy, Precision, Recall, F1-score  
  - Confusion Matrix  
  - Cross-validation  

- **Data Preprocessing**
  - Feature scaling  
  - Handling missing values  
  - Encoding categorical variables
